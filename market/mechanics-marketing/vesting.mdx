# Vesting

Vesting is usually used to lock investors funds and unlock them using time-base function. This means investors still can
use their bound to gain profit by selling it or by using it for governance but can't just sell tokens on the market.

## Actions

As one investor can have more than one vesting contract assigned to him with different schedule we display a filterable
list of available contracts with action buttons. These buttons allow to view contract details, transfer contract
ownership to another account and release vested funds.

## Linear Vesting

Linear vesting is the simplest type of vesting. The amount of assets released over time follows a straight line, meaning the same fraction of the total amount is released every second.

<FunctionPlot options={{
    xAxisLabel: "Days",
    yAxisLabel: "% - Tokens",
    xDomainLimit: 365,
    yDomainLimit: 100,
    data: [{ fn: "x / 365 * 100", color: "#f77423" }] }}
/>

```javascript
const linearVestingConfig = {
  functionType: FunctionType.LINEAR, // LINEAR
  cliff: 0, // No cliff
  duration: 365 * 24 * 60 * 60, // Total duration of 365 days in seconds
  period: 1, // Release every second
  afterCliffBasisPoints: 0, // No immediate release
};
```

### Linear Vesting with Cliff

A cliff can be added to linear vesting, setting a cut-off point before which no assets are released. After the cliff, assets start releasing linearly.

<FunctionPlot options={{
    xAxisLabel: "Days",
    yAxisLabel: "% - Tokens",
    xDomainLimit: 365,
    yDomainLimit: 100,
    data: [
      { fn: "0", range: [0, 91.25], color: "#f77423" },
      { fn: "(x - 91.25) / (365 - 91.25) * 100", range: [91.25, 365], color: "#f77423" },
    ],
  }}
/>

```javascript
const linearCliffVestingConfig = {
  functionType: FunctionType.LINEAR, // LINEAR
  cliff: 91.25 * 24 * 60 * 60, // 91.25 days cliff
  duration: 273.75 * 24 * 60 * 60, // Total duration of 273.75 days after cliff
  period: 1, // Release every second
  afterCliffBasisPoints: 0, // No immediate release
};
```

### Linear Vesting with Immediate Unlock

This type of vesting releases a portion of the assets immediately and then releases the remaining assets linearly over time.

<FunctionPlot options={{
    xAxisLabel: "Days",

    xDomainLimit: 365,
    yDomainLimit: 100,
    data: [{ fn: "25 + (x / 365) * 75", color: "#f77423" }],
  }}
/>

```javascript
const linearImmediateUnlockConfig = {
  functionType: FunctionType.LINEAR, // LINEAR
  cliff: 0, // No cliff
  duration: 365 * 24 * 60 * 60, // Total duration of 365 days
  period: 1, // Release every second
  afterCliffBasisPoints: 2500, // Immediate release of 25% (2500 basis points)
  growthRate: 1n, // Not used in linear vesting
};
```

## Notes

- "afterCliffPercent" can be increased to 100% to implement a simple "Timelock Vesting"

- "period" can be increased to to implement Weekly, Monthly, or Vesting of any custom period in step tranches

- "start" must be passed to schedule vesting activation, cliff period only begins counting after start timestamp
